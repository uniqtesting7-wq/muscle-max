# MuscleMax Mojosari – Gym & Fitness Center

Website resmi **MuscleMax Mojosari**, gym & fitness center terbaik di Mojosari.

## Halaman
- `index.html` – Home
- `about.html` – Tentang Kami
- `programs.html` – Program Latihan
- `schedule.html` – Jadwal Kelas
- `membership.html` – Membership
- `trainers.html` – Trainer
- `gallery.html` – Galeri
- `booking.html` – Daftar Member

## Teknologi
- HTML5, CSS3, JavaScript
- Bootstrap 5
- GSAP (animasi)
- Bootstrap Icons

## Deploy
Website ini di-deploy di **Vercel**.
