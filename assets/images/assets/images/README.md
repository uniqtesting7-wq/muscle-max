## 📌 Cara Ganti Gambar

1. Siapkan foto kamu (format JPG/PNG/WebP)
2. Rename foto sesuai nama file di tabel atas (contoh: `hero-bg.jpg`)
3. **Hapus file `.svg`** yang lama
4. **Taruh foto baru** di folder ini
5. Di HTML, update ekstensi dari `.svg` → `.jpg` (atau format foto kamu)

> 💡 **Tips:** Kompres foto dulu di [squoosh.app](https://squoosh.app) agar website lebih cepat load.
